# FormFlow

FormFlow is a modern, powerful form builder and backend data engine designed to let you build beautiful forms without the hassle. Design stunning, high-converting forms in minutes using an intuitive drag-and-drop builder. Collect data seamlessly, automate your workflows, and engage your audience—no coding required.

## Features

- **Drag-and-Drop Builder**: Easily create forms with various input fields (text, email, long text, checkboxes, etc.).
- **Live Device Preview**: Instantly toggle between Desktop (full width) and Mobile (compact width) preview views while building your form.
- **Form Endpoints**: Submit data from any custom HTML form directly into your FormFlow inbox.
- **Code Snippets Dashboard**: Readily grab embed scripts or async JavaScript fetch examples to plug straight into an external frontend.
- **Comprehensive Dashboard**: View quick form statistics, latest submissions, and your active integration setup all from one workspace.
- **Customization Options**: Adjust primary colors, submit buttons, focus rings, and required field settings in real-time.

## Getting Started

### Prerequisites

You will need [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone or download the repository to your local machine.
2. Navigate into the project directory:
   ```bash
   cd formgrid
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

### Running Locally

Start the Vite development server:

```bash
npm run dev
```

Open `http://localhost:5173` (or the provided local port) in your web browser to view the application.

## Technologies Used

- [React](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
